## 💡 Utilisation

Chaque **Skill** est conçu pour être utilisé avec des assistants IA (Claude, Mistral, ChatGPT, Gemini). Ils fournissent des **réponses structurées, sourcées et conformes au droit français**.

### Commandes rapides (disponibles dans le skill principal)
- `/analyse` : Analyser une situation juridique
- `/recours` : Générer un recours contre une OQTF
- `/admission` : Vérifier l'éligibilité à une admission exceptionnelle
- `/jurisprudence` : Rechercher des décisions pertinentes
- `/courrier` : Rédiger un courrier administratif
- `/caf` : Vérifier l'éligibilité aux prestations CAF/AAH

## 📌 **Comment Utiliser Ces Ressources avec CarthagoLex ?**

1. **Pour les textes juridiques** :
   - Utilisez **[etalab/codes-juridiques-francais](https://github.com/etalab/codes-juridiques-francais)** pour accéder au **CESEDA en Markdown** et l'intégrer dans vos réponses.
   - Exemple : Citez directement les **articles du CESEDA** (ex: `Article L. 511-1 CESEDA`) avec des liens vers Légifrance ou ce dépôt.

2. **Pour la jurisprudence** :
   - Consultez **[GISTI](https://www.gisti.org/)** ou **[Info Droits Étrangers](https://www.info-droits-etrangers.org/)** pour des **analyses à jour** sur les décisions récentes.
   - Exemple : Intégrez des **décisions du Conseil d'État** (ex: `CE, 13 juillet 2018, n° 415478`) dans vos réponses.

3. **Pour les skills IA** :
   - Inspirez-vous de **[lawve-ai/awesome-legal-skills](https://github.com/lawve-ai/awesome-legal-skills)** pour **améliorer vos propres skills** (ex: ajouter des commandes comme `/jurisprudence-cedh`).

4. **Pour les documents juridiques** :
   - Utilisez **[openclaw/skills](https://raw.githubusercontent.com/openclaw/skills/main/skills/hugosbl/legal-docs-fr/SKILL.md)** comme modèle pour **automatiser la rédaction de courriers** (ex: recours gracieux contre une OQTF).
